# Межбиржевой фьючерсный арбитраж (USDT-перпы)

Скрипт ищет межбиржевые расхождения цен по одной и той же бессрочной фьючерсной паре
(USDT-перпы) на пяти биржах — **Binance, Bybit, OKX, Gate, Bitget** — и открывает
дельта-нейтральную пару сделок (шорт на дорогой бирже, лонг на дешёвой), удерживает
её до схождения цен и закрывает с прибылью.

> Реализация по [ТЗ](TZ_futures_arbitrage.md). Только линейные перпы `*/USDT`.
> Никаких прогнозов направления рынка — только межбиржевой арбитраж одного актива.

## Установка

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env      # заполнить реальными ключами
```

## Ключи API

Секреты держатся **только** в `.env` (в `.gitignore`), в `config.yaml` — лишь имена
переменных. Права ключей на старте: **только торговля фьючерсами, без вывода средств**.
Где поддерживается — привязать ключ к IP сервера.

## Конфигурация

Все параметры стратегии — в [`config.yaml`](config.yaml): пороги спреда, комиссии,
плечо, режим маржи, лимиты риска, allow/deny-list тикеров, `dry_run`.

**`dry_run: true`** (по умолчанию) — реальные ордера не отправляются, только логируются
сигналы. Рекомендуется прогнать перед запуском реальных денег (§11).
